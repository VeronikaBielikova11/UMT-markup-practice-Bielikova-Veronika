## Project Description
A web layout practice project created based on a Figma design as part of practical training sessions. The project focuses on developing HTML structure and CSS styling skills.

## Design
The project was developed according to the Figma design:  
https://www.figma.com/design/2Tj16H7IO7dq1ViTvIh57V/Flora?node-id=5999-10563&t=jPjdhtFumlHNz3Xk-1

## Technologies Used
- HTML
- CSS

## Key Features and Implementation
- Interface layout based on a Figma design
- Responsive design for mobile, tablet, and desktop devices
- Mobile-first approach
- Flexbox used for layout structure and element positioning
- Font styling and CSS reset implementation
- Button states (hover, active)
- Modal (popup) windows
- Form handling layout
- Decorative elements and simple animations
- Retina-ready image optimization


**Claude was used to support the development of the dynamic part of the project, including JavaScript implementation.**
